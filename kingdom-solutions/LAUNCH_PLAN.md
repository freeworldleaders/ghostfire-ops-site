# 90-Day Launch Plan (Realistic)

## Day 0–30: MVP foundation
- Web MVP live (Home/About/Creators/FAQ/Start)
- Auth + create story
- Record/upload
- Private-by-default permissions
- Basic export (doc/PDF)
- Trust & Safety + reporting (basic)

Deliverable: Closed alpha (50–200 users)

## Day 31–60: Creator tools
- Opt-in transcription
- Better editor (sections, tags, timeline)
- Export: audiobook script format
- Creator dashboard v1
- Partner pilots (community orgs, libraries, schools)

Deliverable: Beta release + first partner pilot

## Day 61–90: Launch
- Public waitlist -> open access
- Press kit + outreach
- App store prep (if mobile) or PWA packaging
- Monetization pilot (invite-only)

Deliverable: Public launch + measured rollout
